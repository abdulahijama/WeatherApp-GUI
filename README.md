# Weather-App-GUI
## General info
This is a weather forecasting desktop application which retrieves the weather of 10,000 cities using weather API.
	
## Technologies
Project is created with:
* Python: 3.10
* Tkinter
* Weather API
	
## Setup
To run this project, install:
  - Python 3.10
  - Tkinter
  - API Link: "https://openweathermap.org/api"
